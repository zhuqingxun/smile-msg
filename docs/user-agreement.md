## 2. 服务内容

本应用提供即时通讯功能，包括一对一私聊消息的发送与接收。消息采用"阅后即焚"机制，服务端不持久化存储任何聊天记录，服务重启后所有数据自动清除。

我们重视您的隐私。本应用的数据处理原则如下：
